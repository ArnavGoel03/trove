# Trove

A native macOS productivity app — 30+ panes for clipboard, capture, compute, files, system, and storage. Single binary, no telemetry.

**Download:** [latest release](https://github.com/ArnavGoel03/trove/releases/latest) · **Website:** [trove-site.vercel.app](https://trove-site.vercel.app)

Local-only productivity. No accounts. macOS 13+.
